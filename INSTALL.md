# 📲 安装说明书

把 **2026 世界杯预测** 装到你的手机或电脑主屏，像 App 一样打开。

**网站地址**：https://nicolaslinlin.github.io/wc2026/

---

## 📱 iPhone / iPad（Safari）

⚠️ 必须用 **Safari** 浏览器，Chrome 不行。

1. 用 **Safari** 打开网站
2. 点底部中间的 **分享按钮**（方框+向上箭头 ⬆️）
3. 往下滑，找到 **「添加到主屏幕」**
4. 右上角点 **「添加」**

→ 桌面就出现 🏆 图标，点开像 App 一样全屏。

---

## 🤖 Android（Chrome / Edge）

1. 用 Chrome 或 Edge 打开网站
2. 右上角点 **菜单 ⋮**
3. 找 **「添加到主屏幕」** 或 **「安装应用」**
4. 弹窗里点 **「安装」**

→ 主屏出现 🏆 图标。部分手机会自动提示安装，直接点确认即可。

---

## 💻 Windows / Mac 桌面（Edge）

1. 用 Edge 打开网站
2. 右上角点 **菜单 …**
3. 鼠标悬停在 **「アプリ」/「Apps」/「其他工具」** 上
4. 子菜单里点 **「将此站点作为应用安装」** / **「このサイトをアプリとしてインストール」**
5. 弹窗里点 **「安装」**

→ 桌面 + 任务栏出现 🏆 图标。点开是独立窗口，没浏览器地址栏。

---

## 💻 Windows / Mac 桌面（Chrome）

1. 用 Chrome 打开网站
2. 看地址栏**最右边**（在收藏星星附近），找一个**电脑+下载箭头**的小图标 🖥️⬇️
3. 点它 → 弹窗里点 **「安装」**

如果地址栏没看到那个图标，走菜单：
- 右上角 ⋮ → **「投射、保存和共享」** → **「安装 2026 世界杯预测」**

---

## ✨ 装上后能做什么

- 🚀 **打开飞快**（资源已缓存）
- 📴 **离线也能看上次加载的内容**（有网自动刷新）
- 🖼️ **独立窗口**（无浏览器地址栏，像 App 一样）
- 📱 **占用极小**（<1MB）
- 🔔 数据自动更新（每 20 分钟）—— 不需要你手动刷新

## ❓ 卸载

跟卸载任何 App 一样：

- **手机**：长按图标 → 删除/卸载
- **Windows**：开始菜单找到 → 右键 → 卸载
- **Mac**：Launchpad 长按 → ✕

---

## 🐛 遇到问题？

- **iPhone 找不到「添加到主屏幕」**：确认是 Safari，不是 Chrome
- **Android 没有「安装应用」选项**：浏览器需要 Chrome / Edge / Samsung Browser，旧版不支持
- **桌面装了但打不开**：双击图标如果卡住，删了重装一次
